## 四、字体规范

- **中文**: 微软雅黑 (Microsoft YaHei)
- **英文**: Arial

## 六、表格规范

```
- 字体: 11pt, #333333

- 首列(维度列): Bold

列宽: 按内容比例自动分配
```

## 七、布局规则（融合四份PPT风格）

### 科唯可风格 (实战汇报)
- 页面标签使用 BODY 占位符 (20pt)
- 正文使用自定义 TEXTBOX，14-16pt
- 底部可选绿色色块 + 白色结论文字
- 标题即结论，正文展开论据

### 品牌提示物风格 (矩阵布局)
- 场景列：左侧绿色竖条标签
- 对象列：店长/店员/药师/开票员等
- 物品列：具体物品名称
- 成本列：金额/档位
- 评分列：1-10分

### 营销管理风格 (文字密集型)
- 大标题 32-36pt #00B050
- 正文 24pt
- 底部 2.66" 灰色色块锚定
- 无多余装饰

## 八、生成脚本使用

```bash
python3 scripts/generate_simcere.py \
  --template assets/simcere-template.pptx \
  --input content.json \
  --output result.pptx
```

### JSON 内容格式

```json
{
  "slides": [
    {"type": "cover", "title": "...", "subtitle": "...", "dept": "...", "date": "..."},
    {"type": "toc", "items": ["章节1", "章节2"]},
    {"type": "section", "title": "章节标题", "subtitle": ""},
    {"type": "content", "tag": "标签", "title": "页面标题", "body": ["项目列表"], "table": {...}},
    {"type": "review_matrix", "tag": "...", "title": "...", "headers": [...], "rows": [...]},
    {"type": "kpi_dashboard", "tag": "...", "title": "...", "kpis": [...]},
    {"type": "ending", "title": "Thank You", "body1": "...", "body2": "..."}
  ]
}
```
